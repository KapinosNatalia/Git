1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?
1) Чтобы клонировать репозиторий с github.com нужно сначала определиться с папкой на своем локальном компьютере, куда именно будет выполняться клонирование репозитория. Создать папку, если это необходимо, и перейти в нее (в gitBush или терминале).
2) Зайти в соответствующий репозиторий на github.com, нажать на кнопку "code" - "SSH" - скопировать текст из поля.
3) Вернуться в gitBush и ввести команду: git clone "скопированный путь к репозиторию", нажать Enter.


2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
1) Перейти в каталог на своем лок.компьютере, который должен стать репозиторием.
2) выполнить команду "git init" (в текущем каталоге создастся подкаталог ".git" со структурой репозитория)
3) нужно добавить файлы в репозиторий под версионный контроль: "git add ." (или "git add имя_файла")
4) выполнить начальный коммит командой "git commit -m комментарий" 
5) создать репозиторий на github
6) связать локальный репозиторий и репозиторий github командой "git remote add origin "скопированный путь к репозиторию (как в пунтке 2) первого вопроса)""
7) не обязательно, но можно переименовать главную ветку с "master" на "main" командой "git branch -M main"
8) выгрузить данные на github командой "git push -u origin master(или main)"

3. Как получить на локальный компьютер изменения с github.com?
1) командой "git pull"
2) если git не может автоматически выполнить слияние версий локальной и удаленной, и команда "git pull" выдает ошибку, то можно воспользоваться последовательностью команд "git fetch" (получает сведения) и "git merge" (выполняет слияние)

4. Какую информацию нам дают команды:

1) "git status" - показывает, какие файлы каталога в каком состоянии (не отслеживаются, изменены) находятся 
2) "git log" - отображает историю коммитов (автор, дата, комментарий)
3) "git remote -v" - показывает привязку локального репозитория к удаленному репозиторию на github
